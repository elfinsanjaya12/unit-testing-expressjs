# Demo NodeJS REST Api

## Requirement
For running this project please install nodejs and mongodb.
NodeJS:
```
node --version
v12.13.0
```
Postgres:
``
psql --version
psql (PostgreSQL) 11.2
```

## Getting Started
Install npm package:
```
npm install
```

Copy configuration file
```
cp .env.example .env
```

Run the server:
```
npm run start
```

## Run The Test
Copy configuration file
```
cp .env.example .env.test
```

Run test
```
npm run test
```
